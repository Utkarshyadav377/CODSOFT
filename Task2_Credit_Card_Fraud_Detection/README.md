# Credit Card Fraud Detection

## Objective
Detect fraudulent credit card transactions using Machine Learning.

## Dataset
- fraudTrain.csv
- fraudTest.csv

## Algorithms Used
- Random Forest Classifier

## Features
- Data preprocessing
- Encoding categorical features
- Fraud prediction
- Accuracy evaluation
- Confusion Matrix
- Classification Report

## Libraries
- Pandas
- NumPy
- Scikit-Learn

## Run

```bash
python fraud_detection.py
```
